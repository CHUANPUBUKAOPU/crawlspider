本项目作为scrapy爬虫框架复习项目

本项目涉及的知识点有：
1. scrapy爬虫框架
2. scrapy-redis分布式爬虫框架
3. ua（反爬）
4. ip（反爬）
5. redis数据库读取、写入
6. mysql数据库读取、写入
7. pandas库数据管理
8. xpath定位数据
9. git版本控制

本项目开发环境为：
1. 操作系统：ubuntu 22.04.3 LTS
2. miniconda: 23.5.2
    - scrapy: 2.8.0
    - scrapy-redis: 0.7.3
    - pandas: 1.5.3
3. redis-server: 6.0.16
4. mysql: 8.0.33
